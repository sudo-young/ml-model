# ml-model
Basic ML model comparison


# 🧠 ML 모델링 비교 및 클러스터링 분석

## 📌 개요
이 프로젝트는 기초적인 머신러닝 모델의 성능을 비교하고, 클러스터링 기법을 활용하여 데이터의 패턴을 탐색합니다.

## 🛠 사용 기술
- Python (pandas, scikit-learn, matplotlib, seaborn)
- Jupyter Notebook

## 📁 구성
- 회귀/분류 모델 성능 비교 (노트북)
- 클러스터링 분석 (노트북)
- 결과 시각화 파일

## 🔍 분석 주요 내용
- 다양한 모델의 정확도 비교
- 클러스터 중심 데이터 구조 파악
